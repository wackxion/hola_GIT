# Ejemplos Python - Proyecto pequeño

Este proyecto contiene pequeñas demos que muestran las principales mecánicas de Python: variables, control de flujo, estructuras de datos, funciones, clases, excepciones, comprensión, generadores, decoradores y manejo de archivos.

Archivos principales:
- `examples.py`: módulo con funciones demo.
- `main.py`: menú interactivo y ejecución por línea de comandos.

Cómo ejecutar (PowerShell en Windows):

```powershell
cd C:\Users\BRA_W\Downloads\holaGitHub
python main.py         # muestra el menú interactivo
python main.py classes # ejecuta la demo "classes"
```

Notas:
- Requiere Python 3.6+.
- `file_io` escribe un archivo temporal `sample.txt` y lo elimina al terminar.
